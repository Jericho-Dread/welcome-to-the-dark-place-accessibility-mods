# V3 Release

This release includes builds and source for:

- `Dark-Place_Access`
- `Dark-Place_Journal`

## Included Files

- `Dark-Place_Access.Release.zip`
- `Dark-Place_Journal.Release.zip`
- `Dark-Place_Source.zip`

## Requirements

- `Welcome To The Dark Place`
- `BepInEx 5.4.23.5`
- `NVDA` running for speech output

## Accessibility Mod Features

- spoken story text
- spoken choices
- spoken menu focus
- spoken pockets review
- spoken map open/close state
- spoken tape recorder open/close state
- controls help with `F5`
- reread current text with `F6`
- toggle auto choice readback with `F8`
- choose visible options with number keys `1` through `4`

## Journal Mod Features

- create, read, edit, rename, and delete pages
- paste clipboard text into journal entries
- paste current scene text into journal entries
- spoken editing feedback

## Install Summary

### Accessibility mod

1. Install `BepInEx 5.4.23.5`.
2. Copy the `BepInEx` folder from `Dark-Place_Access.Release.zip` into the game folder.
3. Copy `nvdaControllerClient32.dll` and `nvdaControllerClient64.dll` into the game folder beside the EXE.
4. Launch the game with NVDA running.

### Journal mod

1. Make sure BepInEx is already installed.
2. Copy the `BepInEx` folder from `Dark-Place_Journal.Release.zip` into the game folder.
3. Allow the `plugins` folder to merge.

## Main Hotkeys

- `F5`: controls help
- `F6`: reread current text and choices
- `F7`: open or close the journal
- `F8`: toggle automatic choice readback
- `F9`: paste current scene text into the journal while editing
- `F10`: repeat current journal state
- `1` through `4`: activate visible choices from top to bottom
