# Welcome To The Dark Place Accessibility Mods

This repository contains the public source and release packages for:
- `WTTDP.AccessibilityMod` (main accessibility mod)
- `WTTDP.JournalMod` (separate journal mod)

## Downloads
Built packages are in the [`releases`](./releases) folder and can also be attached to GitHub Releases.

Included public packages:
- `WTTDP.AccessibilityMod.Release.zip`
- `WTTDP.JournalMod.Release.zip`
- `WTTDP.Public.Source.zip`

## Notes
- `UndergoingAccess` is intentionally not included in this public repository right now.
- The public accessibility package does **not** bundle BepInEx.
- The accessibility package does include the NVDA controller client DLLs.
- Install BepInEx separately before using these mods.
