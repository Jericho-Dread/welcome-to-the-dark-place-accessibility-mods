# V1.0

This release includes one combined install package:

- `Dark-Place_Access_and_Journal.zip`

## Included In The Package

- `BepInEx 5.4.23.5`
- `Dark-Place_Access.dll`
- `Dark-Place_Journal.dll`
- `nvdaControllerClient32.dll`
- `nvdaControllerClient64.dll`

## Install

1. Copy everything from `Dark-Place_Access_and_Journal.zip` into your `Welcome To The Dark Place` game folder.
2. Allow files to merge or overwrite if prompted.
3. Launch the game with NVDA running.

## Included Mods

### Dark-Place_Access

- spoken story text
- spoken choices
- spoken menu focus
- spoken pockets review
- spoken map open/close state
- spoken tape recorder open/close state
- controls help with `F5`
- reread current text with `F6`
- toggle auto choice readback with `F8`
- choose visible options with number keys `1` through `4`

### Dark-Place_Journal

- create, read, edit, rename, and delete pages
- paste clipboard text into journal entries
- paste current scene text into journal entries
- spoken editing feedback
